# Section Definitions

Rules are organized by filename prefix. Each prefix maps to a section:

| Prefix | Section | Priority |
| --- | --- | --- |
| `foundation-` | Foundation | 1 — HIGH |
| `architecture-` | Architecture | 2 — CRITICAL |
| `integration-` | Integration | 3 — CRITICAL |
| `implementation-` | Implementation | 4 — CRITICAL |
| `delivery-` | Delivery | 5 — HIGH |
